How the Calli Agent Uses These Features

Mode 1: Existing Video Analysis

User Input: Video URL, title, or description of existing content

Agent Process:

  • Extract all applicable features from the video
  • Compute correlations against outcome metrics (views, retention, engagement, CPM, box office)
  • Generate causal hypotheses about performance drivers
  • Report findings with evidence and confidence scores

Output: Feature breakdown, performance analysis, and actionable insights

Mode 2: Prospective Brief Analysis

User Input: Intended video description and story/script outline

Agent Process:

  • Map brief against the feature space
  • Retrieve pre-training priors from billions of comparable content
  • Predict likely performance drivers for the concept
  • Recommend creative adjustments before production

Output: Feature recommendations, risk analysis, optimization suggestions

1 Visual Entry Point

Thumbnail, poster, cover art, or title card analysis. Universal features applicable to YouTube thumbnails, movie posters, streaming covers, and any visual-first introduction to content.

Face Count
face_count
Implemented
Number of human faces prominently visible in the visual.
Computation
Face detection using deep learning (OpenCV, MediaPipe)
Example Values
0, 1, 2-4, 5+
Performance Relevance
Single face = intrigue; multiple faces = ensemble credibility
Documentary: Single expert/subject establishes authority
Comedy Ensemble: 4-6 faces promises group dynamics and humor
Horror: Isolated single face creates dread
Contrast Percentile
contrast_percentile
Implemented
Visual contrast between foreground and background (0-100 scale).
Computation
CLAHE contrast calculation across ROI
Example Values
15, 42, 75, 95
Performance Relevance
High contrast improves thumbnail legibility at small sizes (YouTube 168x94px)
Action: High contrast (80+) emphasizes protagonist against explosion/battle
Horror: Low contrast (20-40) creates fog/obscurity—what can't you see?
Romance: Medium contrast (50-65) balances softness with legibility

Computation

Sci-Fi Spectacle: Higher entropy (5-6) spreads attention across alien landscape
Character Drama: Very low entropy (0.8-1.5) focuses on face and emotion
