# Efstratios Gavves — Professor of Physical AI

A minimal, single-page personal website showcasing research in Physical AI, world models, causal vision, and mechanistic interpretability.

## Features

- **Single-page design** with smooth scroll navigation
- **Interactive hero section** with particle network canvas and TRON-style grid
- **4 Research thrust nodes** with hover cards showing publications
- **Dark theme** with gold accents, inspired by Kira's spatial design
- **Responsive typography** using Space Grotesk for display and Inter for body text
- **Physics-informed animations** including cinematic image pans and interactive grid tilt

## Technology

- Pure HTML5, CSS3, and vanilla JavaScript
- No frameworks or build dependencies
- Optimized for modern browsers
- ~70KB single HTML file with embedded styles and scripts
